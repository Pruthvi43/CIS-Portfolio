Patient Tracker (C++)


A C++ project I made for the purpose of being a tool for medical professionals and aspiring ones that want to practice logging data. It allows for an effective way to log data with interactive components. When designing this program, I planned to give it only one function but later on, I decided to add the game version to increase variety in gameplay. Additionally, when it comes to debugging, I used LLM's and resources like w3schools to figure out the errors in my program. One challenge I overcame was the use of a new concept, structures.
